Отчет анализ рецептур и производства¶

Параметры

  • период (для расч прих с пр-ва)
  • подразделение от кого (для расч прих с пр-ва)
  • подразделение кому (для расч прих с пр-ва)
  • inGoodsGroupId
  • inPriceListId_1 (цены берутся на EndDate)
  • inPriceListId_2 (цены берутся на EndDate)
  • inPriceListId_3 (цены берутся на EndDate)
  • inPriceListId_sale (цены берутся на EndDate)

В гриде - рецептуры показать все (не зависимо от до-та произв), т.е. рецептуры лефт джоин производство

  • Товар
  • Кор рецепт (главный)
  • Кор рецепт (сырье) - zc_ObjectLink_Receipt_Parent
  • расчет с/с для составляющих с ценами прайс-1
  • расчет с/с для составляющих с ценами прайс-2
  • расчет с/с для составляющих с ценами прайс-3
  • кол-во по прих с произв.
  • c/c факт по прих с произв.



Важно - все печати из грида


1.Печатная форма - Рецептуры с разворотом всех составляющих по ценам_1¶

2.Печатная форма - Рецептуры с разворотом всех составляющих по ценам_2¶

3.Печатная форма - Рецептуры без разворота все составляющих по ценам_1, а на те составл что есть рецептуры - по их с/с, т.е. они есть в гриде¶
