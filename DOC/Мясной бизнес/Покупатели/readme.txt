    Подключение надстройки XlsToDBF.

    В EXCEL 2007
1. Кнопка "Office"
2. Кнопка "Параметры Excel"
3. Слева, "Надстройки"
4. Внизу "Управление: Надстройки Excel", кнопка "Перейти"
5. В окне "Надстройки": кнопка "Обзор..."
6. Найти сохранённый ранее файл XlsToDBF.xla и нажать "OK"
7. В окне "Надстройки" в списке доступных надстроек должна появиться строка
   "XLS -> DBF" с установленным флажком проверки, нажать "OK"


    Запуск макроса для сохранения данных в DBF-формате

1. Выделить ячейку относящуюся к экспортируемой области (формат области должен
   отвечать некоторым требованиям, см. ниже "Формат экспортируемой области")
2. Вызвать диалог "Макрос" (ALT-F8)
3. В поле "Имя макроса:" ввести XlsToDBF и нажать "Выполнить"
4. Файл сохранится в той же папке, где расположена открытая книга под именем,
   заданным в первой ячейки экспортируемой области. Если книга только создана и
   ещё не сохранялась, DBF-файл сохраниться в папку по умолчанию (скорее всего,
   в "Мои документы"). Для полной уверенности, перед экспортом имеет смысл сохранить
   книгу.

    Формат экспортируемой области

Программа не включает в себя никаких диалогов настройки экспорта. Поэтому всю
информацию необходимо сообщить непосредственно в заголовке области экспорта.

1. Первая ячейка первой строки области содержит имя выходного DBF-файла (например,
   TEST.DBF)
2. Вторая ячейка первой строки указывает на кодовую страницу для конечного DBF-файла.
   Для кодовой страницы DOS-866 нужно указать CP866 (CP -- разумеется, латиницей).
   При любом другом или пустом значении в этой ячейке конечный файл сформируется
   в кодовой странице WINDOWS-1251.
3  В третьей ячейке первой строки нужно задать значение MAC, если используется русская
   версия MS EXCEL под Macintosh, для корректной перекодировки.
4. Вторая и третья строки содержат описания и названия полей соответственно.
5. Начиная с 4-ой строки и до конца непрерывной области, располагаются данные

Каждое поле ОБЯЗАТЕЛЬНО должно быть описано. Описания полей соответствуют
стандарту dBASE:
    Cnnn - строковый тип (например, C100)
    Nnn или Nnn.n - числовой тип (например, N10 или N10.2)
    D - тип даты
    L - логический тип

Пример:
		Колонка 1:	Колонка 2:	Колонка 3:	Колонка 4:	Колонка:5

Строка 1:	Test.dbf        CP866           MAC
Строка 2:	N10		С20		N8.3            L               D
Строка 3:	ID		Name		Netto           New             D_Prod
Строка 4:	10000		Огурцы		256.5           1               12.01.2013
Строка 5:	20000		Помидоры	113             0               29.11.2012

Логический тип (L) может быть задан, например 0 - False, 1 - True, пустое поле
трактуется, как False. Также в русской версии EXCEL можно использовать "ИСТИНА"
и "ЛОЖЬ".

Файлы SIMPLE.XLS, PRICE.XLS содержат рабочие примеры.
